    About QDrop

    QDrop is a comprehensive open-source QA build distribution platform designed for development teams to upload, manage, and distribute internal builds without relying on Google Play or external services. Built with modern web technologies and cloud-native architecture.

    What is QDrop?

    QDrop provides a streamlined workflow for distributing internal APK and IPA builds to testers. Developers can upload builds through the web interface, attach comprehensive metadata such as version, environment, changelog, and instantly make them available to testers through the companion mobile applications with QR code sharing.

    How It Works

    • • Upload builds using the QDrop Web dashboard with drag-and-drop interface.
    • • APK/IPA files are stored securely on Cloudflare R2 with presigned URLs.
    • • Build metadata is synchronized in Firebase Realtime Database.
    • • Generate QR codes for instant sharing and deep linking via qdrop:// scheme.
    • • QDrop Android/Web apps fetch and display available builds with filtering options.
    • • Testers can download and install builds directly from mobile apps or web pages.
    • • Open access build pages provide distribution links.

    Advanced Features

    • • Real-time Synchronization - Instant updates across all connected devices
    • • Secure Storage - Cloudflare R2 with presigned URLs and access control
    • • Deep Linking - QR codes open directly in QDrop mobile apps
    • • Organization Management - Multi-tenant support with isolated build spaces
    • • Search & Filtering - Find builds by version, platform, or environment
    • • File Integrity - MD5/SHA verification for uploaded builds
    • • Error Handling - Comprehensive error pages and user feedback
    • • Performance Optimized - Lazy loading, caching, and efficient data structures

    Platform Components

    • QDrop Web Dashboard

    Full-featured web interface for uploading builds, managing versions, viewing analytics, organizing QA releases, and generating QR codes for distribution.

    • QDrop Android App

    Native Android app built with Jetpack Compose for browsing, filtering, downloading, and installing APK builds with seamless QR code integration.

    • QDrop iOS Support

    OTA (Over-The-Air) installation support for iOS builds with manifest-based distribution and QR code sharing.

    • Custom Distribution Pages

    Open build distribution pages with QR codes, download links, and branded interfaces for external sharing.

    Open Source & Extensibility

    QDrop is fully open-source and designed to be easily customizable and extensible. The modular architecture allows you to fork the project, modify workflows, integrate with your existing CI/CD pipelines, or extend functionality with custom plugins.

    Public Access URLs

    QDrop provides public URLs for open access to builds and organizations without requiring authentication. Share these links with external testers, stakeholders, or anyone who needs access to your builds.

    Individual Build Pages

    Access specific builds directly via URL. Perfect for sharing individual releases with testers. Set SERVE_PUBLIC_URL=TRUE in your environment variables to enable.

    BASE_URL//organization-id/build-id
